open -a 'Sublime Text' ~/.bash_profile

That should open Sublime
and show you the file .bash_profile.

At the bottom of the file,
on a new line, put the following code:

alias subl="open -a 'Sublime Text' "

Save and close.

and restart Terminal.

HTML

Hyper Text Markup Language

"standard markup language for creating Web pages"
- W3Schools

It's markup (extra stuff) that wraps around the text you see on a page.

It gives meaning to the text in terms of layout and grouping,
and allows us to define element types (text, images, inputs, video, etc.)

To the Terminal!

cd [my code folder]
mkdir class-zero
cd class-zero/
touch index.html
touch styles.css
touch main.js

Open this folder in your text editor

Atom: atom .

Sublime: subl .

TYPE this into index.html:

<!DOCTYPE html>
<html>
  <head>
    <link rel="stylesheet" href="styles.css">
    <script type="text/javascript" src="main.js"></script>
    <title>My Web Page</title>
  </head>
  <body>
    <button id="my-special-button">Turn me red!</button>
  </body>
</html>

TYPE this into styles.css:

#my-special-button {
  color: yellow;
}

TYPE this into main.js:

function turnMyButtonRed(event){
  var button = event.target;
  button.style.background = "red";
}

window.onload = function(){
  var myButton = document.getElementById("my-special-button");
  myButton.addEventListener("click", turnMyButtonRed);
}

Save all files!

Open this page in Chrome

In terminal, type:

open index.html
