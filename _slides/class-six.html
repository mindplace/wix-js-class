The Document Object Model (DOM)

It's how the web pages we view are structured.

The DOM is a tree of nodes, where each HTML element is a node.

A node is connected to its parent node (the node which it is inside of),

and is aware of sibling nodes (nodes which have the same parent).

Why does the DOM matter?

When you target elements, you are actually targeting parts of the DOM.

We even explicitly use the term document.

Lots of times, we target document directly.

Selecting and altering elements

HTML and CSS define the environment within which JS makes changes.

To select / target an HTML element: use its type, class, or ID

Targeting an element with JavaScript

Types, classes, and IDs are written just as they are in CSS

Type example: div

Class example: .my-special-class

ID example: #my-special-id
